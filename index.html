Github Cheatsheet

Configure

git config --global user.name "USERNAME" git config --global user.email "EMAIL"

To see the configuration:

git config --list

Clone & Status

To clone a repository:

git clone LINK

To see files status:

git status

Add & Commit

To add files:

git add FILENAME FILENAME

To commit added files:

git commit -m "MESSAGE"

Push

To push commited files only first time:

git push -u origin main

To push commited files except first time:

git push

Init

To init:

git init

To add a git repository:

git remote add origin LINK

To check the repository:

git remote -v

To check branch:

git branch

To rename branch:

git branch -M main

Branch

To check branch:

git branch

To change branch:

git checkout BRANCH_NAME

To create branch:

git checkout -b BRANCH_NAME

To delete branch:

git branch -d BRANCH_NAME

Merge Branch

To compare with another branch:

git diff BRANCH_NAME

To merge branch:

git merge BRANCH_NAME

Pull

To update local codes from git:

git pull origin main

Undo

To undo add:

git reset FILENAME FILENAME

To undo add (all files):

git reset

To undo commit (1 step back):

git reset HEAD~1

To undo commit (multiple step back):

git reset COMMIT_HASH

To undo commit and backdate vs code:

git reset --hard COMMIT_HASH

To get hash code:

git log
